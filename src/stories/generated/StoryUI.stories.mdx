# Story UI Integration

This Storybook is enabled for AI-powered story generation using Story UI.

- Package: `@tpitre/story-ui`
- Config: `story-ui.config.js`
- Generated stories path: `src/stories/generated/`

To use locally:

1. Run Storybook in one terminal:

```bash
npm run storybook
```

2. Start Story UI in another terminal:

```bash
npm run story-ui
```

Stories created via Story UI will appear under "Ennabl UI/*" in the sidebar and will be included in published builds once committed.

Learn more in the repository: [southleft/story-ui](https://github.com/southleft/story-ui.git).
